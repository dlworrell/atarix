# ATX-SPEC-007 Namespace Model

## Status

Draft wrapper v0.1

## Purpose

This specification formalizes the Atarix Namespace Model.

The current namespace model draft is maintained in:

```text
docs/architecture/namespace-model.md
```

## Normative Source

Until this wrapper is expanded, `docs/architecture/namespace-model.md` is the controlling draft for names, paths, bindings, aliases, lookup, rebinding, unnamed objects, fabric scope, and namespace security separation.

## Required Future Work

- Promote namespace-model.md content into this specification.
- Define exact name syntax.
- Define maximum name and path lengths.
- Define reserved namespaces.
- Define namespace permissions.
- Define namespace cache behavior.

## Dependencies

This specification depends on ATX-SPEC-001 Security Model, ATX-SPEC-002 Authority Model, and ATX-SPEC-006 Object Model.
