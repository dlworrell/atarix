# ATX-SPEC-006 Object Model

## Status

Draft wrapper v0.1

## Purpose

This specification formalizes the Atarix Object Model.

The current object model draft is maintained in:

```text
docs/architecture/object-model.md
```

## Normative Source

Until this wrapper is expanded, `docs/architecture/object-model.md` is the controlling draft for object identity, naming, ownership, ring association, lifecycle, resources, communication, relationships, local objects, and remote objects.

## Required Future Work

- Promote object-model.md content into this specification.
- Resolve open questions about global identity beyond one fabric.
- Define object identity serialization.
- Define object inspection requirements.
- Define object manager responsibilities.

## Dependencies

This specification depends on ATX-SPEC-001 Security Model, ATX-SPEC-002 Authority Model, and ATX-SPEC-004 Lifecycle Model.
