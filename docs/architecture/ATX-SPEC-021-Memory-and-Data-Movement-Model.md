# ATX-SPEC-021 Memory and Data Movement Model

## Status

Draft v0.1

## Purpose

This document begins the Atarix Memory and Data Movement Model.

ATX-SPEC-021 will define memory ownership, data movement forms, buffer descriptors, service-to-service payload movement, accelerator memory access, audit projection, and recovery outcomes.

## Core Rule

Data movement is not authority.

## Initial Movement Forms

- Copy
- Move
- Map
- Share
- Fabric transfer
- Zero-copy optimization
