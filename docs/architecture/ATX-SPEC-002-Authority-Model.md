# ATX-SPEC-002 Authority Model

## Status

Draft stub v0.1

## Purpose

This document will define ownership, control, delegation, revocation, sponsorship, and administrative authority within Atarix.

## Required Topics

- Object ownership
- Authority sources
- Delegation
- Revocation
- Ownership transfer
- Shared control
- Administrative authority
- Bootstrap authority
- Authority audit records
- Authority cleanup

## Required Questions

- Who owns an object?
- Who may delegate authority?
- Who may revoke authority?
- Who may transfer ownership?
- Can authority be inherited?
- Can authority be partially delegated?
- How is authority recovered after crash or reboot?

## Dependency

This specification depends on ATX-SPEC-001 Security Model.
