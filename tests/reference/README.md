# ATARIX Reference Fixtures

This directory is reserved for canonical binary conformance fixtures.

Discovery fixtures are generated by:

```sh
cc -Iinclude tools/mkdiscovery/mkdiscovery.c -o mkdiscovery
./mkdiscovery --profile minimal --output tests/reference/minimal.discovery
./mkdiscovery --profile cpu-card --output tests/reference/cpu_card.discovery
./mkdiscovery --profile supervisor --output tests/reference/supervisor.discovery
./mkdiscovery --profile instrumentation --output tests/reference/instrumentation.discovery
```

The binary fixture files are generated artifacts and should be regenerated from `tools/mkdiscovery/mkdiscovery.c` when the Discovery ROM format changes.

Planned negative fixtures:

```text
invalid_crc.discovery
invalid_length.discovery
duplicate_handle.discovery
unknown_record.discovery
missing_end.discovery
```

Negative fixtures may be generated by a future malformed-profile mode or constructed directly inside unit tests.
