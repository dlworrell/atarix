# ATARIX Public Headers

This directory will contain public C headers generated from or aligned with the ATARIX architecture specifications.

Planned headers:

```text
include/atarix/
├── abi.h
├── identity_eeprom.h
├── discovery.h
├── capability.h
├── service.h
├── bib.h
├── mailbox.h
└── dma.h
```

Headers in this tree should reflect the canonical binary formats documented under `docs/`.
