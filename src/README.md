# ATARIX Source Tree

This directory contains implementation source code for the ATARIX project.

Planned layout:

```text
src/
├── firmware/
│   ├── monitor/
│   ├── bios/
│   └── discovery/
├── kernel/
│   ├── bootstrap/
│   └── services/
├── emulator/
│   ├── fabric/
│   ├── discovery/
│   └── mailbox/
└── common/
```

The architecture and specifications in `docs/` are intended to drive the implementation contained under this tree.
